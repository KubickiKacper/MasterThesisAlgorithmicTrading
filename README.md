# MasterThesisAlgorithmicTrading

### How to run

#### Install backend requirements

`$ cd backend`

`$ python3 -m venv venv`

`$ source ./venv/bin/activate`

`$ pip install -r requirements.txt`

#### Load environmental variables

`$ set -a`

`$ source .env`

`$ set +a`

#### Run backend

`$ flask run`

#### Install frontend modules (2nd terminal)

`$ cd frontend`

`$ npm install`

#### Run frontend

`$ npm run dev`
